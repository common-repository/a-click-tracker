== Description ==

aタグのクリックを全て、Google Analyticsの「イベント」に記録します。

This plug-in record all the clicks of the 'a' tag in 'Event' of Google Analytics.

== Installation ==

1. Upload `a click tracker` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
